Problem 3¶

A change of velocity was described in the earlier chapters as $v_2 = v_1 + a t$. A differential equation would describe it as, $$\frac{ {\tt d}y}{{\tt d} t} = -a t + v_0$$

Use the initial velocity of $v_0 = 0$, $a=-9.8$, initial height of 200, and time points go from 0 to 3.5.

In [4]:
ts = np.linspace(0,3.5);
def model3(init,t):
    v,y = init
    a = -9.8
    dydt = -a*t + v
    y += dydt + 0.5*a*t**2
    return (dydt,y)
ys = si.odeint(model3,[0,200],ts)
plt.scatter(ts,ys[:,1]);
plt.show()

Growth of bacteria initially increases, but then saturation occurs. It is modeled by, $$\frac{{\tt d}P}{{\tt d} t} = k P \left( 1 - \frac{P}{P_{max}} \right) $$ where $P$ is the population count, $k=1$, and $P_{max}$ is the maximum population allowed by the environment.

Model this function for time 0 to 20 with an initial population of 1 and a max population is 10,000.

Model this system and show a plot of your result.

def model5(p,t):

# your code and plot
def model6(init,t):
    x0,y0,z0 = init;
    dxdt = -y0 + 1.8*z0
    dydt = 2*x0 - 2*y0 + 2*z0
    dzdt = -3*x0 - 2.5*y0 + 0.5*z0
    return (dxdt,dydt,dzdt);
init = [-0.3,0.4,0.1];

$$\omega = \frac{{\tt d} (\phi)}{{\tt d} t}$$

Use $g=9.8$ and $R=6$. Initial conditions are $\phi_0=2$ and $\omega_0 = 0$, and the time span is 0 to 20.

Model this function and plot both $\frac{{\tt d} (\phi)}{{\tt d} t}$ and $\phi$ on the same graph.

phi0 = 2;
omega0 = 0;

def model7(inits,t):
    omega,phi = inits;

Problem 8¶

Consider a case of two pendulums which both swing left and right. There is a spring that connects the two masses together. This is called a connected pendulum. (Picture of this on https://en.wikipedia.org/wiki/Pendulum_(mathematics). Scroll down to Coupled Pendula.) This system is described by

$$m \frac{{\tt d}^2 x_1}{{\tt d}t^2} = -\frac{mg}{L} x_1 +k(x_2-x_1)$$$$m \frac{{\tt d}^2 x_2}{{\tt d}t^2} = -\frac{mg}{L} x_2 +k(x_1-x_2)$$

The variables are $m$ is mass, $x_1$ and $x_2$ are the positions of the masses on the end of the pendulum, $k$ is a strength constant, $L$ is the length of each pendulum. Use 100 time steps from 0 to 30. Uses $m=1$ and $k=1$.

The initial conditions are $x_1=0$, $x_2=1$, $\frac{{\tt d}x_1}{{\tt d}t} = 0$ and $\frac{{\tt d}x_2}{{\tt d}t} = 0$.

Model this system and plot $x_1$ and $x_2$ on the same graph.

Hint: If both pendulums start at the same position, then the spring between the masses will never stretch or expand. Thus, you can test your code by starting both masses at the same position and see if their motions are identical.
